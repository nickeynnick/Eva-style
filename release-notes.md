## Ева-стиль 1.0.5

### Автообновление
Установленная версия автоматически проверяет обновления на GitHub при запуске. Также доступна ручная проверка: **Справка → Проверить обновления**.

### Сброс данных
Исправлен сброс данных в десктопной версии. Подтверждение выполняется через поле ввода слова **СБРОС** прямо в панели «Безопасность».

### Резервные копии
Экспорт через кнопку в шапке открывает диалог «Сохранить как…» и позволяет выбрать папку для JSON-файла.

### Файлы релиза
| Файл | Описание |
|---|---|
| `eva-style-setup-1.0.5.exe` | Установщик (рекомендуется, с автообновлением) |
| `eva-style-1.0.5-portable.exe` | Портативная версия (без установки) |
| `latest.yml` | Метаданные для автообновления |
