## Ева-стиль 1.4.0

### Платформа
- **Tauri вместо Electron** — быстрее запуск и меньше нагрузка на слабых компьютерах
- Данные совместимы: «Документы → Ева-стиль → Data»

### Интерфейс
- **Масштаб** — «Владелица» → «Настройки» (85–150%)
- **Проверить обновления** — кнопка в «Справка»

### Производительность
- Часы в шапке не перерисовывают всё окно каждую секунду
- Свёрнутые блоки финансов не держат тяжёлые расчёты в памяти
- Лёгкие иконки в шапке и на приветствии

### Файлы релиза
| Файл | Описание |
|---|---|
| `eva-style-setup-1.4.0.exe` | Установщик (рекомендуется) |
| `eva-style-1.4.0-portable.exe` | Портативный запуск без установки |
