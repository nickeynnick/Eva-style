## Ева-стиль 1.3.0

### Отчёты
- **Надёжная печать PDF** — сводный отчёт за период и «Итог дня» через общий модуль печати (стабильно в Electron)

### Производительность
- **Блоки финансов** — диаграммы и журналы не монтируются все сразу; открываются по клику
- **Диаграммы** — Recharts подгружается при первом раскрытии
- **Иконки** — lucide вынесен в отдельный пакет сборки

### Файлы релиза
| Файл | Описание |
|---|---|
| `eva-style-setup-1.3.0.exe` | Установщик (рекомендуется, с автообновлением) |
| `eva-style-1.3.0-portable.exe` | Портативная версия (без установки) |
| `latest.yml` | Метаданные для автообновления |
