## Ева-стиль 1.2.5

### Отчёты и данные
- **Экспорт CSV + PDF за период** — одна кнопка в сводном отчёте: сводный CSV и печать PDF по выбранному периоду
- **Предпочтительное время автосохранения** — для «раз в день / неделю / месяц» (по умолчанию 18:00)

### Обновления и Windows
- **Статус GitHub** при ручной проверке обновлений
- **Ошибки автообновления** пишутся в CrashLogs
- **Иконка приложения** снова вшивается в `.exe` (ярлык на рабочем столе после переустановки)

### Файлы релиза
| Файл | Описание |
|---|---|
| `eva-style-setup-1.2.5.exe` | Установщик (рекомендуется, с автообновлением) |
| `eva-style-1.2.5-portable.exe` | Портативная версия (без установки) |
| `latest.yml` | Метаданные для автообновления |
