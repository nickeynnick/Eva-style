# История изменений

Все значимые изменения проекта документируются в этом файле.

## [1.1.0] — 2026-07-11

### Добавлено
- **Приветственная анимация** при запуске программы — плавное появление логотипа «Ева-стиль» с исчезновением через 2.5 секунды
- **Список изменений** в диалоге обновления — теперь видно, что нового в обновлённой версии

### Изменено
- **Первый запуск** — список сотрудников теперь пустой, владелица сама добавляет мастеров и администраторов
- **Справка** — удалены интерактивные калькуляторы зарплаты и кассы, удалена информация о кодовом слове для сброса пароля (недоступна публично)

### Исправлено
- **Обновление** — исправлен показ диалога установки после загрузки (`autoInstallOnAppQuit` отключён)
- **Прогресс загрузки** обновления отображается на иконке в панели задач и в заголовке окна
- **Ошибки скачивания** теперь показываются пользователю
- **Скрипт публикации** — исправлен выбор файлов (теперь использует точное имя версии вместо glob)
- **Список изменений** в диалоге обновления теперь отображается корректно

---

## [1.0.5] — 2026-07-11

### Добавлено
- **Автообновление** через GitHub Releases для установленной версии (NSIS)
  - Проверка обновлений при запуске (через 5 секунд)
  - Пункт меню «Справка → Проверить обновления»
- **Портативная версия** — `eva-style-1.0.5-portable.exe` (запуск без установки)
- **Диалог сохранения** резервной копии в десктопной версии (кнопка экспорта в шапке)

### Исправлено
- **Сброс данных** — подтверждение словом «СБРОС» перенесено в интерфейс приложения (в Electron `window.prompt` не работает)
- Сброс данных сразу записывает изменения в `localStorage`
- Версия в меню «О программе» берётся из `package.json` автоматически

### Изменено
- Установщик переименован в `eva-style-setup-1.0.5.exe` для совместимости с автообновлением
- При сборке генерируется `latest.yml` для `electron-updater`

### Примечания
- Автообновление работает только в **установленной** версии, не в portable
- Для автообновления на GitHub Releases должны быть загружены `eva-style-setup-X.X.X.exe` и `latest.yml`
