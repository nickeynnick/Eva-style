# Ева-стиль — Учётный пульт

**Версия 1.1.1** · Десктопное приложение для Windows для управления салоном красоты «Ева-стиль».

Репозиторий: [github.com/nickeynnick/Eva-style](https://github.com/nickeynnick/Eva-style)

## Возможности

- Учёт за день (визиты, касса, расходы)
- Калькулятор услуг
- Солярий
- Зарплаты мастеров и администраторов
- Сертификаты и долги
- Раздел владелицы с настройками и паролем
- Локальное хранение данных на компьютере
- Экспорт и импорт резервных копий (JSON)
- Автообновление через GitHub (установленная версия)

## Установка для пользователя

### Установщик (рекомендуется)

1. Скачайте `eva-style-setup-1.1.1.exe` из [релизов на GitHub](https://github.com/nickeynnick/Eva-style/releases)
2. Запустите файл и следуйте инструкциям мастера установки
3. Запустите «Ева-стиль» с рабочего стола или из меню «Пуск»

Установленная версия автоматически проверяет обновления при запуске.

### Портативная версия

1. Скачайте `eva-style-1.1.1-portable.exe`
2. Запустите файл — установка не требуется

> Портативная версия не поддерживает автообновление. Для обновления скачайте новый файл вручную.

Все данные сохраняются локально на вашем компьютере.

## Резервные копии

- **Экспорт** — кнопка со стрелкой вверх в шапке приложения
- **Импорт** — кнопка со стрелкой вниз в шапке

В десктопной версии экспорт открывает диалог «Сохранить как…».
