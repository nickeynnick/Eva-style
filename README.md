# Ева-стиль — Учётный пульт

Десктопное приложение для Windows для управления салоном красоты «Ева-стиль».

## Возможности

- Учёт за день (визиты, касса, расходы)
- Калькулятор услуг
- Солярий
- Зарплаты мастеров и администраторов
- Раздел владелицы с настройками и паролем
- Локальное хранение данных на компьютере
- Экспорт и импорт резервных копий

## Запуск для разработки

```bash
npm install
npm run electron:dev
```

## Сборка установщика Windows

```bash
npm install
npm run dist
```

Готовый установщик появится в папке `release/`:

- `Ева-стиль Setup 1.0.4.exe` — установщик (рекомендуется)
- `Eva-Style-Portable-1.0.4.zip` — портативная версия без установки
- `win-unpacked/` — распакованное приложение для разработки

## Установка для пользователя

1. Запустите файл `Ева-стиль Setup 1.0.4.exe` из папки `release`
2. Следуйте инструкциям мастера установки
3. Запустите «Ева-стиль» с рабочего стола или из меню «Пуск»

Все данные сохраняются локально на вашем компьютере.
