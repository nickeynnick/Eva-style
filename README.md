# Ева-стиль — Учётный пульт

**Версия 1.2.0** · Десктопное приложение для Windows для управления салоном красоты «Ева-стиль».

Репозиторий: [github.com/nickeynnick/Eva-style](https://github.com/nickeynnick/Eva-style)

## Возможности

- Учёт за день (визиты, касса, расходы, печать итога смены)
- Калькулятор услуг
- Солярий
- Зарплаты мастеров и администраторов
- Сертификаты и долги
- Раздел владелицы с настройками, отчётами CSV и паролем
- Быстрый поиск клиентов, сертификатов и должников
- Локальное хранение данных (единый store с миграциями)
- Экспорт и импорт резервных копий (JSON) с проверкой перед восстановлением
- Автосохранение бэкапа в папку «Документы»
- Автообновление через GitHub (установленная версия)

## Установка для пользователя

### Установщик (рекомендуется)

1. Скачайте `eva-style-setup-1.2.0.exe` из [релизов на GitHub](https://github.com/nickeynnick/Eva-style/releases)
2. Запустите файл и следуйте инструкциям мастера установки
3. Запустите «Ева-стиль» с рабочего стола или из меню «Пуск»

Установленная версия автоматически проверяет обновления при запуске.

### Портативная версия

1. Скачайте `eva-style-1.2.0-portable.exe`
2. Запустите файл — установка не требуется

> Портативная версия не поддерживает автообновление. Для обновления скачайте новый файл вручную.

Все данные сохраняются локально на вашем компьютере.

## Резервные копии

- **Экспорт** — кнопка со стрелкой вниз в шапке приложения
- **Импорт** — кнопка со стрелкой вверх; перед восстановлением показывается предпросмотр
- **Автосохранение** — «Владелица → Безопасность» (папка «Документы → Ева-стиль → Backups»)

В десктопной версии экспорт открывает диалог «Сохранить как…».

## Разработка

```bash
npm install
npm run dev          # веб на http://127.0.0.1:3000
npm run electron:dev # desktop + dev-сервер
npm run lint         # проверка TypeScript
npm run test         # тесты расчётов
npm run build        # сборка фронтенда
npm run electron:build  # установщик Windows (папка release/)
```
